# PBTE

Plataforma de Busca de Transporte Escolar. Sistema web que centraliza a busca por vans escolares no município de Irati-PR, conectando clientes e prestadores de serviço.

Trabalho de Conclusão de Curso do curso de Análise e Desenvolvimento de Sistemas do IFPR.

Autores: Matheus Candido Xavier e Igor Michel Mazo.
